# Reports

## Responsabilidad

Consolida informacion de stock, gastos, ventas, senas, saldos, inversion y rentabilidad.

## Archivos principales

- `components/ReportsPanel.tsx`
- `queries.ts`
- `services/`
- `types.ts`
- `constants.ts`

## Rutas que lo usan

- `/admin/reportes`

## Tablas Supabase

- `vehicles`
- `vehicle_expenses`
- `sales`

## Se puede modificar aca

- metricas
- resumen financiero
- reportes por vehiculo
- calculos globales

## No modificar desde aca

- carga primaria de vehiculos
- carga primaria de gastos
- carga primaria de ventas

## Producto futuro

Es un modulo de valor para productos Pro/Full. Puede depender de varios modulos porque su funcion es consolidar.
