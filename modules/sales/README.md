# Sales

## Responsabilidad

Gestiona reservas, senas, ventas, entregas, cancelaciones, saldo pendiente y estado comercial de operaciones.

## Archivos principales

- `components/SalesPanel.tsx`
- `queries.ts`
- `services/`
- `types.ts`
- `constants.ts`

## Rutas que lo usan

- `/admin/ventas`
- `/admin/reportes`

## Tablas Supabase

- `sales`
- actualiza `vehicles.status` cuando corresponde

## Se puede modificar aca

- carga de operaciones
- estados de venta
- senas
- saldos
- cancelaciones

## No modificar desde aca

- ficha publica del vehiculo
- notas de CRM
- configuracion general

## Producto futuro

Es parte de `AndiCars Pro` y `AndiCars Full`.
