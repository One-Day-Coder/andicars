# Public Site

## Responsabilidad

Contiene componentes publicos comerciales que no pertenecen exclusivamente al stock ni al CRM.

## Archivos principales

- `components/PublicContactInfo.tsx`
- `components/WhatsAppButton.tsx`
- `constants.ts`

## Rutas que lo usan

- `/`
- `/autos/[id]`

## Tablas Supabase

- `app_settings`

## Se puede modificar aca

- datos publicos de contacto
- boton de WhatsApp
- secciones comerciales generales

## No modificar desde aca

- formulario de consulta si crea leads
- catalogo o ficha tecnica del vehiculo

## Producto futuro

Es parte de `AndiStock`, `AndiCars Pro` y `AndiCars Full`.
