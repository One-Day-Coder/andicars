# AndiCars - hoja de ruta

Este archivo guarda mejoras recomendadas para hacer mas adelante. La idea es no mezclar todo de golpe y avanzar cuando el negocio lo necesite.

## Hacer despues de probar con consultas reales

- Asignar consultas a un vendedor especifico.
- Crear vista tipo embudo con columnas: Nuevo, Contactado, Interesado, Negociando, Perdido.
- Exportar consultas/clientes a Excel.
- Agregar origen editable para consultas cargadas manualmente: web, WhatsApp, Instagram, referido, local, MercadoLibre.

## Hacer cuando haya varios usuarios trabajando

- Permisos mas finos por accion, no solo por modulo.
- Auditoria mas completa: guardar version anterior de cada nota editada.
- Historial de actividad por usuario.

## Hacer antes de usar con mucho publico

- Revisar textos legales y privacidad de datos.
- Agregar captcha si llega spam real.
- Mejorar SEO del catalogo y fichas de autos.

## Hacer cuando el flujo comercial ya este claro

- Modulo de tareas/agenda semanal.
- Recordatorios por email o WhatsApp.
- Reportes por vendedor y por origen de consulta.
- Dashboard de conversion: consultas, reservas, ventas y perdidas.

## Hacer al final o cerca del lanzamiento formal

- Ajuste estetico completo.
- Dominio propio.
- Revision mobile fina.
- Fotos optimizadas y textos finales de la agencia.
